Problem areas and possible solutions

Home landing page

  • User wants to be able to sort by a specific type of bike
  • User wants to look at bikes that are on sale
  • User wants to look at mens/womens bikes
  • User wants to search for keywords in a search bar
  • User needs to trust the quality of the website they visit
  • UI: Dropdown tabs for bike variants
  • UI: Sections on the home page for bike types with headers
  • UI: A quiz or filter option to find a fit
  • UI: A search bar at the top of the page

Product grid page

  • User wants to sort the bikes by price, gender, color
  • User wants some information on that type of bike. A summary
  • User wants to be able to filter bikes by some simple specifications
  • UI: A simple but robust filter option
  • UI: A summary of this variant of bike/the type of tab clicked on

Product page

  • User wants to look at many high quality photos
  • User wants to look at reviews from other customers
  • User wants to see the size of the bike, see if it will fit
  • UI: A Photo carousel or photo section for viewing many possible angles of the bike, zoom in for details
  • UI: a choice to swap through different colors of the bike in the photos
  • UI: A prominent CTA for leaving reviews and for viewing others' reviews. Incentive to leave a review
  • UI: Height selection option, weight of the bike
